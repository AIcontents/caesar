# Проект "Шифр Цезаря"

Этот проект представляет собой консольное приложение на Java для шифрования и дешифрования текста с использованием шифра Цезаря.

## Функциональные возможности

Приложение поддерживает следующие операции:

1.  **Шифрование:** Шифрует текстовый файл с помощью ключа.
2.  **Дешифрование:** Дешифрует текстовый файл с помощью ключа.
3.  **Дешифрование методом Brute-force:** Подбирает ключ шифрования путем перебора всех возможных вариантов.
4.  **Дешифрование методом статистического анализа:** Дешифрует текст, анализируя частоту встречаемости символов в зашифрованном тексте.

## Как использовать

1.  Склонируйте репозиторий.
2.  Соберите проект с помощью Maven.
3.  Запустите приложение из командной строки.
4.  Следуйте инструкциям в консоли, чтобы выбрать режим работы и указать пути к файлам.
